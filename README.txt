# Transoceanica Vehiculos

Bienvenido a Transoceanica Vehiculos, donde puedes explorar y comprar una variedad de vehículos.

--Descripción

Este código implementa un sistema simple de compra de vehículos. Muestra una lista de productos, permite al usuario elegir si desea comprar y agregar productos al carrito. Al finalizar, muestra el carrito y el total a pagar.

--Instrucciones de Uso

1. Al ejecutar el código, se dará la bienvenida a los usuarios a Transoceanica Vehiculos.
2. Se solicitará al usuario que indique si desea comprar alguna unidad respondiendo con "si" o "no".
3. Si el usuario elige comprar ("si"), se mostrará la lista de unidades disponibles.
4. El usuario podrá agregar unidades al carrito especificando el tipo y la cantidad.
5. El proceso continuará hasta que el usuario decida no agregar más unidades ("no").
6. Finalmente, se mostrará el carrito con los productos seleccionados y el total a pagar.

--Ejemplo de Ejecución

1. Ejecutar el código.
2. Responder "si" al prompt inicial.
3. Seleccionar unidades y cantidades.
4. Responder "no" cuando se pregunte si se desean más unidades.
5. Se mostrará el carrito y el total a pagar.
